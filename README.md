> Detect if Mocha is running

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i detect-mocha --save
```

## Usage

```js
var detectMocha = require('detect-mocha');
if(detectMocha()) {
  // doSomethingFancy
}
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
